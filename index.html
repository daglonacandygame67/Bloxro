🔌 RobbieBux & Ads Integration Guide

Games run in a sandboxed iframe and talk to RobbieBlox using window.postMessage. Every published game automatically gets a small RobbieAds helper injected (see "🤖 Add Reward Ads with Claude" on the Create page) — you don't need to hand-write the message plumbing below unless you want to.

Rewarded ads — grant your own in-game reward when a player opts in:

RobbieAds.showRewarded(watched => {
  if (watched) { /* extra life, bonus points, unlock… */ }
});

RobbieBux — read balance, spend, or open a shop, by posting messages to window.parent:

// Ask for the player's balance
parent.postMessage({type:'robbieblox:getBalance'}, '*');
// Listen for: {type:'robbieblox:balance', balance}

// Spend RobbieBux (credited to you as the creator)
parent.postMessage({type:'robbieblox:spend', amount:10, itemId:'sword', itemName:'Sword'}, '*');
// Listen for: {type:'robbieblox:spendResult', success, newBalance}

// Open a full shop UI
parent.postMessage({type:'robbieblox:openShop', title:'🛍️ Shop', items:[
  {id:'sword', name:'Sword', icon:'⚔️', price:25}
]}, '*');

Reminder: rewarded ads must always be opt-in (a button the player chooses to tap), never a requirement to keep playing — see the Terms of Service.
